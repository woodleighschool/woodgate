# Privacy Policy

The app does not use advertising, analytics, or tracking technologies.

The app communicates only with the backend server configured for its operation. The app developer does not provide a public cloud service for this app.

Any data transmitted by the app is sent only to the configured backend service used to operate the app and is handled under the control of that backend operator.

If you have questions about data handling for a specific deployment, contact the organisation operating that backend service.
